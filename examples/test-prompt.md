# Test Prompt

Use this prompt to test Skill Efficiency Reviewer against another Codex skill.

```txt
Use $skill-efficiency-reviewer to review this skill:

/Users/your-name/.codex/skills/website-safety-auditor/SKILL.md

Check clarity, reusability, workflow logic, practicality, output quality, scalability, and maintainability.
Keep the review concise and operator-friendly.
```

You can also paste a skill directly:

```txt
Use $skill-efficiency-reviewer to review the Codex skill below.

[paste skill here]
```
